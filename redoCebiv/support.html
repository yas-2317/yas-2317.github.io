解析が失敗・途中で止まった場合、クレジットは戻りますか？
はい、自動的に返金されます。 AI処理の失敗・保存エラーなど、サービス側の原因による場合、消費されたクレジットは自動的に返金されます。返金はリアルタイムで反映されます。

解析が「処理中」のまま長時間経過している場合は、お問い合わせフォームよりご連絡ください。
対応しているスタックを教えてください
現在対応しているスタックは以下の通りです：

Web: Next.js / React / TypeScript / Tailwind CSS / Vue / Nuxt / Svelte / SvelteKit
Mobile: Swift / SwiftUI / Flutter
Backend: Ruby on Rails / Python

対応スタックは順次拡大予定です。
ZIPファイルに制限はありますか？
アップロード前に node_modules / .git / build成果物 を除外してください。これらが含まれると解析精度が下がり、クレジットが無駄になります。

目安: ソースファイルのみのプロジェクトは通常1〜5MB程度です。
プランの違いを教えてください
Wanderer（無料）: 月3クレジット付与
Tracer（$5/月）: 月80クレジット付与
Navigator（$12/月）: 月250クレジット付与

クレジット消費の目安：
初回解析（ZIPアップロード）: 5cr / トレース生成: 1cr / 変更候補: 2cr / 課題採点: 無料
サブスクリプションのキャンセル方法は？
アカウント設定の「サブスクリプション管理」からいつでもキャンセルできます。キャンセル後は現在の期間終了まで有料プランをご利用いただけます。

App Storeからは管理できません。
アップロードしたコードはどこに送られますか？
解析処理のため、コードの内容はAnthropicのAPIに送信されます。送信されたコードはサービス提供のみに使用され、学習・広告目的での利用はされません。

ZIPファイル自体は解析完了後に削除されます。コードファイルは最終アクセスから30日後に自動削除されます。
30日間使わないとどうなりますか？
最後にプロジェクトを利用した日から30日後に、コードファイルの内容が自動削除されます。

削除後は新しいトレース・変更候補の生成ができなくなりますが、過去のトレース履歴・課題結果・成長トラッキングは引き続きご覧いただけます。

再度ご利用になる場合は、ZIPを再アップロードするだけで解析を再開できます（5cr必要）。

FAQs

If analysis fails or gets stuck, will my credits be refunded?
Yes, automatically. If analysis fails due to a Service-side issue (AI error, save failure, etc.), consumed credits are automatically refunded in real time.

If your project has been stuck in "analyzing" for a long time, please contact us via the form below.
What stacks are supported?
Currently supported stacks:

Web: Next.js / React / TypeScript / Tailwind CSS / Vue / Nuxt / Svelte / SvelteKit
Mobile: Swift / SwiftUI / Flutter
Backend: Ruby on Rails / Python

More stacks are being added regularly.
Are there any restrictions on the ZIP file?
Before uploading, please exclude node_modules, .git, and build artifacts. Including them reduces analysis accuracy and wastes credits.

A typical source-only project is around 1–5 MB.
What's the difference between plans?
Wanderer (Free): 3 credits/month
Tracer ($5/month): 80 credits/month
Navigator ($12/month): 250 credits/month

Credit costs: Initial analysis (ZIP upload): 5cr / Trace generation: 1cr / Change suggestion: 2cr / Challenge grading: Free
How do I cancel my subscription?
You can cancel anytime from "Subscription Management" in your account settings. After canceling, you retain access to your paid plan until the end of the current billing period.

Subscriptions cannot be managed through the App Store.
Where does my uploaded code get sent?
Your code is sent to Anthropic's API for analysis. It is used solely to provide the Service and is never used for training or advertising purposes.

ZIP files are deleted after analysis completes. Code file contents are automatically deleted 30 days after your last use of the project.
What happens if I don't use a project for 30 days?
Code file contents are automatically deleted 30 days after your last use of the project.

After deletion, new traces and change suggestions cannot be generated, but your past traces, challenge results, and progress tracking remain accessible.

To resume, simply re-upload the ZIP file (5 credits required).
